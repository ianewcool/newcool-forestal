# NewCool Forestal - CONAF Chile

Portal educativo sobre el sector forestal y CONAF en Chile.

## Demo

[https://forestal.newcool.io](https://forestal.newcool.io)

## Caracteristicas

- Buscador de oficinas CONAF
- Guia de permisos forestales
- Catalogo de especies nativas
- Calculadora de plan de manejo
- Normativa forestal

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
